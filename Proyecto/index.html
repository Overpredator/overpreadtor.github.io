Actividad Recreativa en Pareja

Realizar una infografía en pareja es una actividad eficaz para fortalecer la confianza mutua.

Además, fomenta el desarrollo de habilidades cognitivas y estimula la creatividad en la resolución de problemas.

Recursos para la actividad

Para llevar a cabo esta actividad, pueden consultar los siguientes enlaces que ofrecen información sobre HTML y la estructura de las páginas web:

  • Estructura de un documento HTML
  • Cómo se estructura una página web
  • Partes de una página web: Estructura y contenido
  • Estructura básica de un documento HTML

Juntos, elaboren una infografía que resuma la estructura básica de una página web, utilizando la información proporcionada en los enlaces.

Esta actividad les permitirá desarrollar sus habilidades y creatividad en el diseño y comprensión de páginas web.
